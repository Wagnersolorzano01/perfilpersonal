Perfil Personal con Formulario de Contacto usando PHP y MySQL

Este proyecto tiene como finalidad desarrollar un sitio web personal que incluya información del usuario y un formulario de contacto funcional en PHP para el envío de mensajes.

El objetivo principal es aplicar conocimientos en HTML, CSS, PHP y el despliegue de un proyecto en un hosting gratuito.
Objetivo del Proyecto

Comprender y aplicar conceptos relacionados con:

Desarrollo web con HTML5 y PHP
Diseño responsivo utilizando Bootstrap
Validación de formularios en cliente y servidor
Conexión con bases de datos MySQL mediante PHP PDO
Almacenamiento seguro de datos
Organización de archivos web
Publicación de un sitio web en hosting gratuito
Funcionalidades del Sistema

El sistema incluye las siguientes funcionalidades:

Página principal con información personal
Biografía corta del estudiante
Sección de hobbies e intereses
Formulario de contacto
Validación de nombre, correo y mensaje
Confirmación de envío del formulario
Registro de mensajes en base de datos
Visualización del historial de mensajes registrados

XAMPP
PHP
MySQL

Instalación del Proyecto
Descargar o clonar el repositorio.
Copiar la carpeta del proyecto en:

C:\\xampp\\htdocs



Iniciar Apache y MySQL desde XAMPP.
Abrir phpMyAdmin.
Crear una base de datos.
Importar el archivo .sql correspondiente del proyecto.
Configurar el archivo conexion.php con los datos de conexión a la base de datos local.

Ejemplo:

Host:
localhost

Usuario:
root

Contraseña:
(vacía por defecto en XAMPP)

Abrir el navegador y ejecutar:

http://localhost/nombre-del-proyecto/
